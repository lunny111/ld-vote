# ld-Vote
Vote System for Mayor Elections [UI] [FiveM]

## Dependencies
- ld-ui
- mythic_notify (https://github.com/JayMontana36/mythic_notify)

## How to Setup

1 - Start all scripts in your server.cfg

2 - You can edit candidates in ld-vote/config.lua
